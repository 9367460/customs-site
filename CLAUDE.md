# Customs Site — Таможенный консалтинг

## Описание проекта
Статический сайт таможенной компании (HTML + CSS + JS).

## Структура файлов
- `index.html` — главная страница
- `about.html` — о компании
- `customs.html` — таможенное оформление
- `auto-shipping.html` — автоперевозки
- `rail-shipping.html` — ж/д перевозки
- `sea-shipping.html` — морские перевозки
- `payments.html` — таможенные платежи
- `ved-outsourcing.html` — ВЭД аутсорсинг
- `style.css` — стили
- `script.js` — скрипты
